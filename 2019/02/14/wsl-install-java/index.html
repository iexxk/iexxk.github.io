    win10子系统Ubuntu安装java

    安装java环境
    配置环境变量

    sudo vim ~/.bashrc在文件末尾添加

    执行source ~/.bashrc生效配置

    测试java -version

    运行tomcat

    进入tomcat安装目录，windos和linux通用，然后右键打开ubunt，执行./catalina.sh run

    注意事项

    tomcat不要放Program Files带空格或特殊字符的路径，在linux下识别不了

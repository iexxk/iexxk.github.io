    提示

    解决原因，是网络配置文件uuid冲突，导致不能上网，修改即可

    执行uuidgen ens33生产新的830a6ae2-85fb-41e7-9e5d-60d084f56f5f替换配置文件里面的

    执行nmcli con | sed -n '1,2p'进行验证

    eureka高可用(未实践)

    启动:

    java -jar app.jar --spring.profiles.active=peer1

    java -jar app.jar --spring.profiles.active=peer2

    注意：

    1. 高可用启动，application.name一定要一致
    2. 一定要指定端口号
    3. 不能禁用自我注册，注意配置文件加载顺序及覆盖顺序
